Introduction

YALLA is Yet Another Logging Library Abstraction for .NET.

Figure 1: Deployment diagram

Supported Platforms

  • .NET Framework 4
  • Silverlight 5
  • Windows 8 (WIP)
  • Windows Phone 8.1 (WIP)
  • Windows Phone Silverlight 8 (WIP)
  • Xamarin.Android (WIP)
  • Xamarin.iOS (WIP)
  • Xamarin.OSX (WIP)
  • Portable Class Libraries

 Note

This is a preliminary release with limited functionality. .NET Framework 4 (or later), Silverlight 5 and Portable Class Libraries are currently the only supported platforms.

Getting Started

As shown in Figure 1 above, YALLA supports a variety of use cases, which include the following:

  1. Application Code Input: Your application code either requires the addition of logging functionality, or offers logging functionality that requires modification.

  2. Library Code Input: Your library code either requires the addition of logging functionality, or offers logging functionality that requires modification.

  3. Platform Facility Output: Your application or library code requires logging functionality with one or more platform-specific facility outputs.

 Note

This is a preliminary release with limited functionality.Application Code Input, Library Code Input and Platform Facility Output are the only use cases that are currently supported.

You only need to add the YALLA Core Library as a dependency to your project in order to apply the Application Code Input use case or the Library Code Input use case. The YALLA Core Library documentation contains installation instructions and relevant API reference.

In order for your application to produce YALLA log(s), you need to add YALLA as a dependency to the main project(s) (The YALLA Core Library dependency will be added for you.). If you haven't already, install the Yalla NuGet package. The current version is Version 0.1.3.

The logging system must be properly initialized at application startup. The recommended method is Configure.

The logging system must be properly terminated at application teardown. The recommended method is Shutdown.

Detailed YALLA API information may be found in the Reference section.
